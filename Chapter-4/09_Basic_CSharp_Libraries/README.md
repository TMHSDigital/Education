# Basic C# Libraries

## Math

```csharp
public class Example {
    public static void Main(string[] args) {
        Console.WriteLine(Math.Sqrt(16));  // Square root
        Console.WriteLine(Math.PI);        // Pi constant
    }
}
```

## Date

```csharp
using System;

public class Example {
    public static void Main(string[] args) {
        DateTime now = DateTime.Now;
        Console.WriteLine(now);
    }
}
```
