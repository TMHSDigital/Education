# Collections

Ruby provides powerful collection types: Arrays, Hashes, and Ranges.

## Arrays

```ruby
fruits = ["apple", "banana", "cherry"]
fruits << "orange"
puts fruits
```

## Hashes

```ruby
ages = {"John" => 30, "Jane" => 25}
ages["Tom"] = 20
puts ages
```

## Ranges

```ruby
range = 1..5
range.each do |num|
  puts num
end
```
