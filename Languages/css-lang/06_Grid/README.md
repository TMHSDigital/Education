# Grid

CSS Grid Layout is a two-dimensional layout system for the web. It lets you layout items in rows and columns.

## Basic Grid Properties

### Display Grid

```css
.container {
  display: grid;
}
```

### Grid Template Columns

```css
.container {
  grid-template-columns: repeat(3, 1fr);
}
```

### Grid Gap

```css
.container {
  grid-gap: 10px;
}
```
