# Functions

## Defining Functions

```javascript
function greet(name) {
}
```

## Function Arguments

```javascript
function add(a, b) {
    return a + b;
}

console.log(add(2, 3));
```

## Return Statement

```javascript
function square(x) {
    return x * x;
}

let result = square(4);
console.log(result);
```
