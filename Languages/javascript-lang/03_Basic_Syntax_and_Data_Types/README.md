# Basic Syntax and Data Types

## Variables and Data Types

```javascript
// Integer
let x = 5;

// Float
let y = 3.14;

// String
let name = "JavaScript";

// Boolean
let isActive = true;
```

## Basic Operators

```javascript
// Arithmetic operators
let a = 10;
let b = 5;
console.log(a + b);  // Addition
console.log(a - b);  // Subtraction

// Comparison operators
console.log(a > b);  // Greater than
console.log(a === b); // Equal to
```
