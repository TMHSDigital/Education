# Control Structures

## Conditional Statements

```javascript
let x = 10;
if (x > 5) {
    console.log("x is greater than 5");
} else if (x === 5) {
    console.log("x is 5");
} else {
    console.log("x is less than 5");
}
```

## Loops

### For Loop

```javascript
for (let i = 0; i < 5; i++) {
    console.log(i);
}
```

### While Loop

```javascript
let count = 0;
while (count < 5) {
    console.log(count);
    count++;
}
```
