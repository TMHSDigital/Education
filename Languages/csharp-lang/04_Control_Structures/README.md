# Control Structures

## Conditional Statements

```csharp
int x = 10;
if (x > 5) {
    Console.WriteLine("x is greater than 5");
} else if (x == 5) {
    Console.WriteLine("x is 5");
} else {
    Console.WriteLine("x is less than 5");
}
```

## Loops

### For Loop

```csharp
for (int i = 0; i < 5; i++) {
    Console.WriteLine(i);
}
```

### While Loop

```csharp
int count = 0;
while (count < 5) {
    Console.WriteLine(count);
    count++;
}
```
