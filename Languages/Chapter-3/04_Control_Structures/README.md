# Control Structures

## Conditional Statements

```java
int x = 10;
if (x > 5) {
    System.out.println("x is greater than 5");
} else if (x == 5) {
    System.out.println("x is 5");
} else {
    System.out.println("x is less than 5");
}
```

## Loops

### For Loop

```java
for (int i = 0; i < 5; i++) {
    System.out.println(i);
}
```

### While Loop

```java
int count = 0;
while (count < 5) {
    System.out.println(count);
    count++;
}
```
