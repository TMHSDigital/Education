# Data Structures

## Lists

```python
fruits = ["apple", "banana", "cherry"]
print(fruits[0])  # Accessing elements
fruits.append("orange")  # Adding an element
print(fruits)
```

## Tuples

```python
person = ("John", 30)
print(person[0])
```

## Dictionaries

```python
student = {"name": "Alice", "age": 25}
print(student["name"])
student["grade"] = "A"
print(student)
```

## Sets

```python
colors = {"red", "green", "blue"}
colors.add("yellow")
print(colors)
```
