# Methods

Methods in Ruby are a way to group reusable code into a single unit.

## Defining Methods

```ruby
def greet(name)
  puts "Hello, \#{name}!"
end

greet("World")
```

## Method Arguments

```ruby
def add(a, b)
  a + b
end

puts add(2, 3)
```

## Return Statement

```ruby
def square(x)
  x * x
end

result = square(4)
puts result
```
