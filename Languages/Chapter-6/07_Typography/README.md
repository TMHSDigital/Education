# Typography

Typography involves the styling of text on your web page. CSS provides several properties to control the appearance of text.

## Basic Typography Properties

### Font Family

```css
body {
  font-family: Arial, sans-serif;
}
```

### Font Size

```css
h1 {
  font-size: 2em;
}
```

### Font Weight

```css
h1 {
  font-weight: bold;
}
```

### Text Align

```css
p {
  text-align: justify;
}
```
