# Basic Syntax and Data Types

R is known for its simple and expressive syntax. It supports various data types and basic operations.

## Variables and Data Types

```r
# Numeric
x <- 5

# Character
name <- "R"

# Logical
is_active <- TRUE

# Vector
numbers <- c(1, 2, 3, 4, 5)

print(x)
print(name)
print(is_active)
print(numbers)
```
