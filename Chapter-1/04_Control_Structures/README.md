# Control Structures

## Conditional Statements

```python
x = 10
if x > 5:
    print("x is greater than 5")
elif x == 5:
    print("x is 5")
else:
    print("x is less than 5")
```

## Loops

### For Loop

```python
for i in range(5):
    print(i)
```

### While Loop

```python
count = 0
while count < 5:
    print(count)
    count += 1
```
