# Collections

Swift provides powerful collection types: Arrays, Sets, and Dictionaries.

## Arrays

```swift
var fruits = ["apple", "banana", "cherry"]
fruits.append("orange")
print(fruits)
```

## Sets

```swift
var colors: Set = ["red", "green", "blue"]
colors.insert("yellow")
print(colors)
```

## Dictionaries

```swift
var ages = ["John": 30, "Jane": 25]
ages["Tom"] = 20
print(ages)
```
