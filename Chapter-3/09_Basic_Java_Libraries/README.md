# Basic Java Libraries

## Math

```java
public class Example {
    public static void main(String[] args) {
        System.out.println(Math.sqrt(16));  // Square root
        System.out.println(Math.PI);        // Pi constant
    }
}
```

## Date

```java
import java.util.Date;

public class Example {
    public static void main(String[] args) {
        Date now = new Date();
        System.out.println(now);
    }
}
```
